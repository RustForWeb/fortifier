# Introduction

Schema validation.

- Synchronous & asynchronous validation
- Enums & structs
- Typed errors
- Built-in validations
    - Email address
    - Length
    - Phone number
    - Range
    - Regular expression
    - URL
- Support for Serde & Utoipa
